Das Plugin optimiert einen Shop auf regionale Lieferungen. Hierzu erzwingt es eine PLZ-Angabe und stellt ein Lieferzeit-Feld bereit.

== Description ==

Das Plugin ermöglicht es kleinen Restaurants, einen WooCommerce Shop auf regionale Lieferungen zu optimieren. Ziel ist,
kleinen Restaurants mit einfachen Mitteln eine Chance zu geben, regionale Lieferungen anzubieten.

Die Funktionsweise wird in [diesem Blogbeitrag](https://binary-butterfly.de/artikel/tutorial-low-budget-shop-mit-regionaler-lieferung/)
genauer erläutert.
